# SeminarRegistrationApp
## Deskripsi
Aplikasi android sederhana untuk registrasi seminar menggunakan kotlin.
## Fitur
- Input data peserta
- Navigasi antar halaman
- Tampilan UI sederhana
## Teknologi
- Kotlin
- Android studio
- ## Cara Menjalankan
  1. Clone repository
  2. Buka di android studio
  3. Jalankan di emulator
